- species_alignment.zip: CDS alignment between Dmel, Dsim, Dyak, Dsan, Dtei used for ancestral reconstruction

- Dmel_Anc_alignments.zip: CDS alignment between Zambia population and Dmel/Dsim ancestor used to create consensus sequences

- Dmel_Anc_consensus_sequences.zip: consensus sequences used to run MASS-PRF

- MASS-PRF_output: contains raw results of MASS-PRF that run succesfully 

- MASS-PRF_tables_only contains table of interest from the MASS-PRF results. 

- scripts: contain pipeline and scripts used for the analysis

- results: contains MASS-PRF profiles of protein with adaptive clusters and a list of experimentally tractable candidates

- iso1_correspondence_table.tsv: Correspondence between the names of iso1 transcript and the OG ID
